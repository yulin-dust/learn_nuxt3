#### 1. 前置知识
```angular2html

多页面页面，每访问一个新页面。都会加载所有的资源。

客户端渲染，响应之后在浏览器进行渲染
服务端渲染，响应的时候就已经渲染好了数据

Ajax 数据客户端渲染，因为是从客户端进行修改的

npm install 安装
npm run dev 启动 （在 package.json 文件中： "dev": "nuxt dev",）
```

npx nuxi@latest init <project-name>
